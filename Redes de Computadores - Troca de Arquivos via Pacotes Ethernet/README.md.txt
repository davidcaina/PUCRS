# Trocar de pacotes via IPv4 - /UDP/TCP & IPv6 - /UDP/TCP

## Aplicação:
- Esse projeto consistiu no envio e recebimento de pacotes (arquivos de texto - .txt, no caso) através dos protocolos IPv4 e IPv6.
- Foi desenvolvimento como projeto final para a cadeira de Redes III no curso de Ciências da Computação.
