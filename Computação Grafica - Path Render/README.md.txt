# Computação Grafica - Path Render

## Aplicação:
- Durante um periodo do dia, foi filmado o caminho que os alunos, do predio 32 do campus, fizeram até a saida do predio. As filmagens foram convertidas em coordenadas e utilizadas para o desenvolvimento de um jogo.
- Foi desenvolvimento como projeto final para a cadeira de Computação Grafica, no curso de Ciências da Computação.
